# VariablesYOperadores
Este es un programa realizado con Variables y Operadores en Java
# ¿Para qué se utilizan los operadores lógicos en programación?
* Se utilizan para verificar si una o mas condiciones se cumplen o para hacer preguntas complejas; generalmente se utilizan con un if-else.
# ¿Por qué es importante declarar correctamente el tipo de dato de una variable?
* Para que cuando se soliciten el usuario pueda ingresar correctamente los datos sin que le de ningun tipo de error en el programa, ya que si se declara un tipo de dato y en la pantalla a el usuario no se le especifica que dato debe ingresar, podria ingresar datos que no estan permitidos en el programa.
## Justificacion de Incisos 4-6
*  Inciso 4: Utilice un "if-else", porque me permite analizar el rago de la condicion dada y me indica en que categoria se encuentra el número ingresado.
*  Inciso 5: Utilice un "Switch", porque me permitio crear y organizar mi menu con un case por cada dia, ya que tenia multiples opciones para la misma variable.
*  Inciso 6: Utilice un "if-else", porque me permite ingresar el usuario y contraseña, y me dice: si el usuario y la contraseña son correctos, entonces escribe "acceso concedido", sino "contraseña incorrecta"; y si el usuario no coicide, muestra "usuario no registrado"
